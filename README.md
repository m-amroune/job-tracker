# Job Tracker

Job application tracker built with Next.js.

---

## About the Project

### Objective

Build a simple tool to track job applications locally.

The project focuses on :

- State management
- Local data persistence
- Clear user interactions
- Maintainable structure

---

## Project Overview

The tracker includes the following features:

- **Application Creation**  
  Add a company and a position using a controlled form.

- **Application Editing**  
  Inline editing of company and position fields.

- **Status Workflow**  
  Click on the status to cycle through the application stages (todo → applied → interview → rejected).

- **Deletion**  
  Remove an application with confirmation.

- **Persistence**  
  All data is stored in the browser using localStorage.
